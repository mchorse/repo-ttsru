# TTSRu

A simple client-side mod for turning Russian Cyrillic characters into English phonetics while using chat, i.e. "Привет" turns into "Preeveht". The algorithm is pretty primitive, but overall it's quite understandable.

Простой клиентский мод который превращает русскую кириллицу в английскую фонетеку во время использования чата, к примеру, i.e. "Привет" turns into "Preeveht". Алгоритм не идеален, но в целом всё понятно.
